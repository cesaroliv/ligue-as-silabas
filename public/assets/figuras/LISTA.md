# Figuras das palavras — lista de produção

Coloque os PNGs **nesta pasta** (`public/assets/figuras/`).

- Formato: PNG 512×512 com fundo transparente
- Nome: exatamente como abaixo (minúsculas, sem acento)
- Não precisa mexer em código: o jogo detecta os arquivos sozinho
  (recarregue a página depois de copiar)
- Enquanto uma figura não existe, o jogo mostra o nome da palavra escrito
  (fallback provisório)
